CityBike

This is my travel app design. I illustrated the app to create a prototype design. The app is designed for tourists who need to locate bikes for rent. I wanted to create something really unique and bright for an enhanced user experience.

These screens below show the users experience when using the app. The screens are very bright and have a unique style.

For my prototype I designed different icons for the choosen UK cities. The user can select which city they would like to find a rental bike or they will have the option to choose 'current location'.
